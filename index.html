    모임기간 : 2월 1일 (목) 12시 00분 ~ 18시 00분
    모임장소 : 서울 광진구 구의동 성진빌딩 3층 메트로비즈니스

    CSS 능력을 중고급으로 향상 시키기 위해서는 float, position, display, overflow 속성에 대한 확실한 이해가 없이는 불가능합니다.
    이런 속성들은 혼자 공부하기에는 어려운 속성들임은 분명합니다.
    하지만 핵심 이론 수업과 이론을 활용한 실정 웹 페이지를 제작 수업을 하면서 어렵기만 했던 float, position, display, overflow 속성을 활용해서 div 레이아웃 설계 방법을 확실히 이해하실 수 있습니다.
    요소들을 마음대로 배치하기 위해서 반드시 알아야 할 flot 속성
    요소를 감추고 보이게 하기 위해 반드시 알아야 할 position 속성
    요소의 성질을 자유롭게 바꾸기 위해서 반드시 알아야 할 display 속성
    float으로 배치된 요소의 부모요소의 높이 값을 자동으로 조절하기 위해서 반드시 알아야 할 overflow 속성
    레이아웃 설계를 위해 반드시 알아야 할 div 수평정렬/수직정렬
    TML+CSS를 잘 한다고 말할 수 있는 단계까지 올라가는건 쉬운 일이 아닙니다.
    하지만 스스로 공부해서 수준급으로 갈 수 있도록 도약대가 되는 개념과 원리를 충분히 이해할 수 있도록 이해하기 쉽게 강의합니다.
    특히, 실무에서 정말 필요한 기능과 노하우를 담은 웹표준 HTML5 & CSS3 정석을 알려드리겠습니다.
